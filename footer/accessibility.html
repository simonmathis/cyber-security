Accessibility Features

At Cyber Security, we are committed to providing an inclusive and user-friendly experience for all visitors. We continuously work to ensure our website meets accessibility standards and provides features that enhance usability for everyone, including individuals with disabilities.

Available Accessibility Features

Dark/Light theme toggle

We offer a dark and light theme toggle to accommodate different visual preferences and improve readability. This feature allows users to select the theme that best suits their needs, reducing eye strain and improving overall accessibility.

Keyboard Navigation

Our website is designed to be fully navigable using a keyboard. Users can move between interactive elements using the Tab key and activate links and buttons using the Enter key.

Screen Reader Compatibility

We strive to ensure our website is compatible with popular screen readers. Our content is structured with proper HTML semantics and ARIA attributes to provide an optimal experience for visually impaired users.

Scalable Text and Zoom Support

Users can adjust text size using their browser’s zoom settings without loss of content or functionality. We ensure that text remains legible and adaptable across different screen sizes and resolutions.

Alt text for images

All meaningful images on our website include descriptive alternative text (alt text) to assist users who rely on screen readers.

Contrast and Readability

We design our website with clear typography and high-contrast color schemes to improve readability. The dark/light theme toggle further enhances this by providing options suitable for different lighting conditions.

Continuous Improvement

We are dedicated to improving our website’s accessibility. If you encounter any accessibility issues or have suggestions for enhancements, please contact us at SC238252@bexhillcollege.ac.uk. Your feedback helps us create a better and more inclusive experience for all users.
